# Blockchat

A simple blockchain-based chat application. Decentralized, communal, cool as fuck.

## Features

- Decentralized chat application using blockchain technology.
- Each message is stored as a block in the blockchain.
- Peer-to-peer communication.
- Automatic blockchain synchronization between peers.

## Installation

1. Clone the repository:

2. Install the required dependencies:
     pip install -r requirements.txt
 
## Usage

1. Start a node:
    python node.py <port>

2. Enter your username when prompted.

3. Connect to other nodes by running the same command on different ports.

## Commands

- `/exit`: Exit the application.
- `/help`: Display the help message.
- `/ping`: Check the connection to peers.
- `/list`: List all connected peers.
- `/blockchain`: Display the latest block in the blockchain.
- `/history`: Display the chat history.
- `/save`: Save the current blockchain and peer list to disk.
- `/load`: Load the blockchain and peer list from disk.
- `/clear`: Clear the console.
